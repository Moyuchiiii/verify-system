# Correctness Discovery

情報を「正確」と判定する前に、以下の6項目を必ずチェックする。

## 6項目チェックリスト

1. **この情報を使うのは誰か**
   - 対象読者・用途によって「正確」の基準が変わる

2. **いつ役立たなくなるか**
   - 時限性のある情報（統計・法律・価格等）は取得日時を明記する

3. **いつ有害になるか**
   - 正確でも文脈次第で誤解を生む可能性がある情報を特定する

4. **不確実性をどう扱うか**
   - 「確実」「可能性が高い」「推測」を明確に区別して表記する

5. **リスク耐性**
   - 誤りを見逃すことと、正確な情報を「誤り」と判定することのどちらが許容できないか

6. **検証方法**
   - この情報が正確であることを独立したソースで確認できるか

## 適用タイミング

- verifierが判定を下す前に必ずこのチェックリストを通過させる
- document-reviewerが総合評価を出す前にも確認する
