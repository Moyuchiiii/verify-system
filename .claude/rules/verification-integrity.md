# Verification Integrity

裏取り・ファクトチェックの誠実性を保つための強制ルール。

## 絶対禁止

- MUST NOT: ソースURLなしに「✅ 検証済み」と判定する
- MUST NOT: WebSearchを実行せずに「確認した」と報告する
- MUST NOT: 「たぶん正しい」「おそらく」で判定を終わらせる
- MUST NOT: investigatorの証拠収集が完了する前にverifierが判定を出す

## 必須事項

- 検証に使った検索クエリは全件記録する（省略禁止）
- ソースが確認できない情報は必ず「🔍 未検証」とする
- 一次ソース（公式文書・査読論文等）と二次ソースを区別して記録する
- 判定に使用したソースのアクセス日時を記録する

## 完了条件

以下をすべて満たさない限り、検証タスクを「完了」としない:

1. investigatorが証拠収集を完了し報告した
2. verifierが競合仮説分析を完了し判定を出した
3. 全クレームに判定（✅/⚠️/❌/🔍）とソースが付いている
4. document-reviewerが総合スコアを算出した
