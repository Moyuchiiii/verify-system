# pdf-scan

## 使い方

```
/pdf-scan [PDFファイルパス]
```

例:
- `/pdf-scan input/pdfs/article.pdf`

## 実行フロー

1. **PDF読み込み**: `pdf`スキルを使ってPDFの全文を取得する
2. **クレーム抽出**: 以下の要素を重点的に抽出する
   - 数値・統計データ
   - 日付・時系列情報
   - 固有名詞（人名・企業名・地名）
   - 因果関係の主張（「〜が原因で〜になった」等）
   - 引用・参照している外部情報
3. **fact-checkに引き渡す**: 抽出したクレームリストをfact-checkスキルに渡す

## 注意事項

- クレームが20件を超える場合は重要度の高いものから優先して処理する
- 著者の意見・主観的記述はファクトチェック対象外とし、明示的にスキップする
